# Product Manager — Checklist

引用 [00-agent-framework.md](../../03-agents/00-agent-framework.md) §13 Checklist。

---

## 工作前检查

□ 是否已读取 Project Standard？

□ 是否已读取 Agent Standard？

□ 是否已读取 Document Standard？

□ 是否已读取 Workflow Standard？

□ Task 和 Workflow 是否已从 AI Project Manager 接收完整？

□ 需求分析报告是否已从 Requirement Analyzer 获取？

---

## 工作中检查

### PRD 阶段

□ 是否使用了 [03-prd-template.md](../../06-templates/03-prd-template.md) 模板？

□ 背景 / 问题 / 目标 / 用户故事 是否已明确？

□ 功能需求是否按优先级（P0 / P1 / P2）排列？

□ 验收标准是否可测试？

□ 是否覆盖了非功能需求？

□ 是否分析了 API 和数据库影响？

□ 是否识别了风险？

### 设计阶段

□ 产品设计方案是否已考虑技术可行性？

□ 是否与 Architect 沟通了技术方案？

### 验收阶段

□ 功能是否满足 PRD 定义？

□ 非功能需求是否达标？

□ 验收结论是否已记录？

---

## 工作后检查

□ PRD 是否已交付给 Architect / Engineer？

□ 验收结论是否已交付给 AI Project Manager？

□ 所有交付物是否使用了正确模板？

□ Metadata 是否完整（Version / Status / Owner / Last Updated）？

□ 是否已通知下游 Agent？

---

## 交付物检查

□ PRD 包含：背景与目标 / 用户故事 / 功能需求 / 非功能需求 / 验收标准 / 影响范围 / 风险

□ 验收标准使用 Given / When / Then 格式

□ 功能需求有优先级
