# Release Manager — Checklist

引用 [00-agent-framework.md](../../03-agents/00-agent-framework.md) §13 Checklist。

---

## Release 前检查

□ 所有 Code Review 是否已通过？

□ QA 测试是否已通过？

□ 产品验收是否已通过？

□ 数据库迁移是否已测试？

□ 配置是否已更新到目标环境？

□ Release Note 是否已准备？

---

## Release 中检查

□ 部署是否成功？

□ 服务是否正常启动？

□ 监控是否正常？

---

## Release 后检查

□ 是否已验证核心功能？

□ 是否需要 Rollback？

□ Release Note 是否已发布？

□ 是否已通知相关方？
